HuerTechno

Proyecto de huerto tecnologico que tiene por objetivo automatizar el riego, la luz, humedad y temperatura, además generar un registro para posterior analisis y estudio. Se desea que el sistema sea lo suficientemente flexible para controlar cualquier especie de planta.

El proyecto se está elaborando con sensores de bajo costo tales como DHT11 para temperatura y humedad relativa, yl-69 para humedad de suelo y una foto celda para la luminosidad; nos obstante, está pensado para instalar cualquier tipo de sensor.

El trabajo consta de dos partes:

  • Mediciones
  • Control y analisis

Mediciones

En esta etapa el sistema debe ser capaz de recoger los datos mediante los sensores que están controlados por una placa de arduino y son transmitidos a una raspberry donde son almacenados.

Control y analisis

En la segunda etapa, se debe poder ingresar al sistema en la raspberry y controlar las variables del sistema, poder regar y controlar la luz en forma manual y/o automática.

El proyecto tiene fines educativos, no comerciales y está licenciado bajo software libre
